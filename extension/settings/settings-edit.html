Configure Languages:

Enter new languages you want to upload models for here. A language's identifier is a unique pair of letters used to identify the language's models and the name is the name of the language.

The languages defined here determine what languages are present in the dropdowns.

Upload New Models:

Upload new models here; download them from Firefox Translations Models. Each model should come with a model file, lex file, and one or more vocabulary files.

Translations can only be performed if there is a model from one language to the other.
